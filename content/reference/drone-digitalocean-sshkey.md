---

A string field, provides the fingerprint of the ssh keypair that you
wish to add to the server's authorize_keys file upon creation.

```
DRONE_DIGITALOCEAN_SSHKEY=3b:16:bf:e4:8b:00:8b:b8:59:8c:a9
```
