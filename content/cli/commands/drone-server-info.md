This subcommand prints individual server information. Please note this command requires administrative privileges.

Example usage:

```
$ drone server info i-f0e4c2f76c58916ec

Name:    i-f0e4c2f76c58916ec
Size:    m5.xlarge
Region:  us-east-1
Address: 198.51.100.245
State:   running
```
